# Borderline Web

Web version of [Borderline](https://apps.apple.com/app/id6772781356) — real MLB pitch calls in your browser.

**Live:** https://azzopalj.github.io/borderline-web/

## Stack
- React 18
- Vite
- Canvas API for pitch animation
- Baseball Savant (Statcast) for pitch data

## Dev
```bash
npm install
npm run dev
```

## Deploy
Push to `main` — GitHub Actions builds and deploys automatically.
